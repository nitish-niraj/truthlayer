# TruthLayer — Submission Package

This directory is a self-contained, portable copy of the TruthLayer
final submission. Everything needed to evaluate the project lives here
— no external dependencies beyond the live URLs listed below.

---

## What's inside

```
submission/
├── README.md                  # Production-quality project overview
├── PROJECT_LINKS.md           # All live URLs in one place
├── FINAL_SUMMARY.md           # Architecture, testing, lessons learned
└── docs/
    ├── SCREENSHOTS.md         # Walkthrough with captions
    ├── DEMO.md                # Demo video reference
    ├── screenshots/           # 9 full-resolution PNGs
    └── demo/
        └── truth-layers.mp4   # 90-second product walkthrough
```

---

## How to use this package

1. Open `README.md` for the full project overview, architecture, and
   installation steps.
2. Open `FINAL_SUMMARY.md` for the architecture summary, test results,
   deployment details, and lessons learned.
3. Open `PROJECT_LINKS.md` for every live URL.
4. Open `docs/SCREENSHOTS.md` for the screenshot walkthrough.
5. Open `docs/DEMO.md` to find the demo video, or play it directly
   from `docs/demo/truth-layers.mp4`.
